# Marss Real Estate

A modern and responsive front-end for a real estate website, designed to provide a seamless user experience for browsing, buying, and selling properties.

## Features

*   **Dynamic Home Page:** A captivating home page with a hero video and a curated list of featured properties.
*   **Property Listings (Buy/Rent):** Dedicated pages for buying and renting properties, complete with advanced filtering options available directly in the navigation.
*   **Sell Your Property:** A user-friendly form for property owners to submit their properties for sale.
*   **Informative Pages:** "About Us" and "Contact" pages to provide company information and contact details.
*   **Loan Calculator:** An integrated loan calculator to help users estimate their mortgage payments.
*   **Dynamic Navigation & Footer:** Navbar and footer components are loaded dynamically across all pages for consistency and ease of maintenance.
*   **Responsive Design:** A fully responsive layout that works on all devices, from desktops to mobile phones.

## Technologies Used

*   **HTML5:** For the structure and content of the website.
*   **CSS3:** For styling and creating a modern look and feel.
*   **JavaScript (ES6+):** For interactive elements, dynamic content loading, and form handling.
*   **Bootstrap 5:** For the responsive layout and UI components.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

You will need a modern web browser to view the project.

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/iamrealvinnu/DEVGREL.git
    ```
2.  Navigate to the project directory
    ```sh
    cd DEVGREL
    ```
3.  Open the `Frontend/index.html` file in your browser to view the website.

For a better development experience, you can use a live server to automatically reload the page when you make changes. Many code editors have extensions for this, such as "Live Server" for VS Code.

## Usage

The website is designed to be intuitive and easy to use. Users can navigate through the different pages using the navigation bar. The "Buy" and "Rent" sections (accessible via the "Properties" dropdown) allow users to filter properties based on location, type, and configuration. The "Sell" page provides a form for users to submit their property details. The "Loan Calculator" helps users estimate their monthly loan payments.
